# wld脚本

`wld脚本` 是一个学习通 / 雨课堂用户脚本维护版，主文件为 `wld脚本.js`。

## 当前保留功能

- 学习通与雨课堂视频、音频自动播放
- PPT、文档、电子书、图文任务自动阅读
- 学习通任务点与章节自动跳转
- 学习通章节页遇到未完成作业时自动跳过当前作业任务点并继续后续流程
- 作业页和考试页允许粘贴
- 雨课堂常见防切屏检测拦截
- 自动查题、自动答题、AI/OCR 与外接题库功能已全部移除


## 使用说明

1. 在 Tampermonkey 或同类用户脚本管理器中导入 `wld脚本.js`。
2. 进入学习通或雨课堂对应课程页面后，脚本会自动识别并运行。
3. 可在脚本面板中调整雨课堂倍速和 PPT 翻页间隔。
